#  Конвертер валют ЦБ РФ
Приложение для конвертации валют с актуальными курсами Центрального банка России.
##  Технологии

- **Flutter** - фреймворк для разработки
- **Dart** - язык программирования
- **http** - для запросов к API ЦБ РФ
- **xml** - для парсинга XML ответов
### Требования
- Flutter SDK (>=3.0.0)
- Android Studio / VS Code
- Git
URL: https://www.cbr.ru/scripts/XML_daily.asp
Формат: XML
Запускайте через flutter run
